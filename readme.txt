The Easy and Lightweight Plugin to add custom CSS to your wordpress website. This plugin will help you to override Theme or Plugin CSS. 

== Description ==
Custom CSS for Wordpress Plugin allows you to add custom CSS to your Wordpress Website. By using this plugin you can easily override Theme or Plugin CSS. 

= Features =
 
* Easy to Use
* Line Numbers indicatated for each CSS lines
* Find Option using (ctrl+F)

= How to add Custom CSS? =
 
* Go to Appearance Menu > Select Custom CSS Sub Menu.
* Enter your Custom CSS here.
* Click on "Save Custom CSS" Button. That's it!

If you need any help. You can add ticket in Support section. Suggestions will be welcomed.

== Installation ==
1. Download the plugin and extract the files
2. Upload 'custom-css-wp' to your '~/wp-content/plugins/' directory
3. Activate the plugin through the 'Plugins' menu in WordPress
